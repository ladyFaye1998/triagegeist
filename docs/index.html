How It Works

This demo simulates the ensemble model's decision process using a rule-based approximation of the trained LightGBM + XGBoost + CatBoost stacked pipeline. The actual model uses 300+ features including TF-IDF NLP trigrams — this interactive version uses the top clinical predictors.

ComponentDetails
ModelsLightGBM + XGBoost + CatBoost → LR meta-learner, 5-fold stratified CV
Features250+ (vitals, demographics, NLP, comorbidity history, qSOFA, SIRS)
OOF Performance99.59% accuracy, 0.9980 QWK (stacked ensemble)
Bias Analysis5 demographic dimensions, intersectional subgroup analysis
